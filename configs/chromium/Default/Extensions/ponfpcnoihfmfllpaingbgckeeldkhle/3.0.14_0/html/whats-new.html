What's new in Enhancer for YouTube™ v3.0.14

Below is important information that requires your attention!

Bugs

  • Fixed all volume and speed control issues caused by recent UI changes on YouTube.
  • Fixed issues with the pop-up player. To resolve them, videos must now be loaded from the “www.youtube-nocookie.com” website (owned by YouTube). As a result, the extension now requires permission for this site. However, if you don’t use the pop-up player, you can safely remove this permission. To do so, open a new tab and paste this URL: chrome://extensions/?id=ponfpcnoihfmfllpaingbgckeeldkhle
    Then, under the “Site access” section, uncheck the permission.

Experiments

Enhancer is not yet compatible with the new experimental player controls, so by default, the extension blocks them until YouTube phases out the old controls. However, many users want to use the new controls. If it’s your case, you can enable them, but only by configuring Enhancer to allow experiments and ensuring that Enhancer’s controls are placed outside the player (i.e., in a control bar).

Unlike other features, this can’t be set in the options page, as the information must be retrieved immediately when the page starts loading, and your preferences aren’t available at that stage. That's why we must use a cookie. More info here if you want to allow experiments, under the “YouTube experiments” section. If you run into issues, please don’t report them—there’s a reason some experiments are blocked.
